Voicecopter
===========

Our cincy node copter project.  It takes voice commands from a laptop and executes them on the drone.

## Commands

* Launch
* Land
* Elevate
* Lower
* Advance
* Back
* Left
* Right
* Shake
* Dance
* Flip
* Fire Missles
* Reset

## Requirements

This requires node and sox with flac support.
